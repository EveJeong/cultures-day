# 비주얼 디자인 — "보노보노 무지개 PPT" 컨셉

> 앱 전체의 시각 언어. 2000년대 한국 워드아트(WordArt) 감성: 무지개 그라데이션 배경 +
> 두꺼운 테두리 글자 + 보노보노 마스코트 + 병맛 유머. 참조 이미지 기반.
> 관련: `03-display-design.md`(이 톤으로 렌더), `02-admin-design.md`.

## 0. 브랜딩

- **프로젝트/행사명**: **"바딧불이 야호"**
- **로고**: 누끼 딴 **보노보노 이미지 + "바딧불이 야호" 워드아트** 조합.
  - `closed`(로고) 화면·헤더·로딩에 사용.
- **메인 페이지**: 보노보노 무지개 배경 위에 **"바딧불이 야호"** 워드아트 타이틀.
- Remote Config `event_name` 기본값 = `"바딧불이 야호"`(`04-data-model.md` §2).

## 1. 컨셉 한 줄

**"촌스러운데 신난다"** — 행사 분위기를 띄우는 과장된 무지개 PPT 미학.

## 2. 컬러

| 토큰 | 값(예시) | 용도 |
| --- | --- | --- |
| 무지개 배경 | 가로 그라데이션 `#FF00CC → #5B6CFF → #00E0FF → #33E060` (최대 채도) | 전 화면 기본 배경(애니메이션 가능) |
| 워드아트 옐로 | 채움 `#FFEE00` + 테두리 `#FF0000` | 메인 제목(예: 채널명) |
| 워드아트 블루 | 채움 `#1A1AE6` + 테두리 `#FF0000` | 강조 제목(예: 제목) |
| 워드아트 화이트 | 채움 `#FFFFFF` + 옅은 테두리 | 소제목 |
| 푸터 톤 | 배경에 살짝 묻히는 저대비 | "구독과 좋아요" 류 장난 문구 |

## 3. 타이포그래피

- **두껍고 둥근 한글 디스플레이 폰트**: Jua / Do Hyeon / 검은고딕 / BM여린체 류.
- **워드아트 효과**: 굵은 글자 + `text-stroke`(외곽선 2~4px) + drop-shadow + 살짝 3D 느낌.
- 제목은 옐로/블루 채움 + 빨강 외곽선 조합으로 대비 강조.
- 장식: `*^^*`, `★`, 물결, 무지개 이모지 등 과장된 데코 허용.

## 4. 마스코트

- **보노보노**(파란 바다수달) 캐릭터를 비어있는 공간·환영 화면에 배치.
- 사용처: `closed`(로고) 화면, 로딩, intro 화면 코너, 빈 상태(empty state), result 축하.
- 에셋: `assets/bonobono.png`(누끼 딴 투명 PNG, §7).
- ⚠️ 보노보노는 저작권 캐릭터(이가라시 미키오) — **사내 1회성 행사 내부 사용** 전제. 외부 배포/상업 사용 지양.

## 5. 컴포넌트 톤

| 요소 | 스타일 |
| --- | --- |
| 버튼 | 두툼·둥근 모서리·외곽선·살짝 그림자. 호버 시 통통 튀는 모션 |
| 패널/카드 | 반투명 흰/검 패널로 가독 영역 확보(무지개 위 텍스트 대비) |
| 점수 강조 | 워드아트 숫자 + 카운트업 애니메이션 |
| 전환 | 무지개 슬라이드·반짝임 등 과장된 트랜지션(과하지 않게) |

## 6. 가독성 균형 (중요)

전 화면을 풀 무지개로 도배하면 **빔프로젝터의 점수·문제 가독성**이 떨어진다. 원칙:

- **축하/연출 표면**(로고·intro·result·리더보드 헤더)은 컨셉 풀로 — 화려하게.
- **정보 표면**(점수 숫자·문제 텍스트·운영 컨트롤)은 **반투명 패널 + 고대비 텍스트**로
  가독성 우선. 무지개는 배경/테두리/헤더에서 살리고, 읽어야 하는 본문은 깔끔하게.
- 운영 페이지(`/admin`)는 조작 정확성이 중요하므로 컨셉은 **헤더·악센트 수준**으로 절제.

## 7. 누끼(배경 제거) 작업

참조 이미지에서 보노보노 캐릭터만 투명 PNG로 추출한다.

- 도구: Python `rembg`(U²-Net 딥러닝 세그멘테이션) — 무지개 같은 복잡 배경도 캐릭터 분리.
- 입력: `assets/source/<참조이미지>` (사용자가 디스크에 저장).
- 출력: `assets/bonobono.png`(투명 배경).
- 절차: `pip install rembg onnxruntime` → 스크립트 실행(모델 최초 1회 다운로드).
- 대안: 깔끔한 보노보노 공식/투명 PNG를 직접 구해 쓰면 누끼 불필요.
